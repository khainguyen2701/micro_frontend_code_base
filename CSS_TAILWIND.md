### Tailwind CSS

Hiện tại chúng ta đang sử dụng Tailwind CSS để style các component trong micro frontend.
Đối với các remote application, chúng ta chỉ có thể dùng ở phạm vi remotes, không thể dùng ở phạm vi shell và chia sẻ cho các remote được tại vì tailwind css không hỗ trợ chia sẻ style giữa các remote application.

Hiện tại chưa tìm được cách nào để chia sẻ style giữa các remote application, mặc dù đã có sử dụng cách đọc file theo path, nhưng vẫn gặp phải lỗi không thể sử dụng style của remote application trong shell.
